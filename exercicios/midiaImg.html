Trabalhando com a tag img - DIO

A tag HTML img é utilizada para incorporar uma imagem em uma página web. É uma tag vazia (contém apenas os atributos e não possui tag de fechamento), sendo dois desses atributos obrigatórios:

  1. src: especifica o caminho para a imagem
  2. alt: especifica o texto alternativo para a imagem
É possível especificar a altura e a largura da imagem, com os seguintes atributos:
  1. width: define a largura de uma imagem
  2. height: define a altura de uma imagem
