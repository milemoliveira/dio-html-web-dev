Guia Básico - Tags HTML

O que é HTML?

HTML significa HyperText Markup Language, traduzindo Linguagem de Marcação de Hipertexto.

Afinal, o que isso quer dizer? O HTML não é uma linguagem de programação. É uma linguagem de marcação, usada para definir a estrutura do seu conteúdo.


Elementos essenciais do HTML

  • !DOCTYPE: instrução para o navegador web dizendo em qual versão do HTML a página é escrita.
  • html: representa a raiz de um HTML, todos os outros elementos devem ser descendentes desse elemento.
  • head: elemento que contém meta informações sobre a página HTML, não é o conteúdo mostrado para quem vê a página.
  • title: define o título da página (mostrado na barra de guias do navegador).
  • body: é o cortpo do documento, contém todos os conteúdos visíveis, como títulos, parágrafos, links, listas, imagens etc.
